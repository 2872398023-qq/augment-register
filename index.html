🚀 Augment Register API

这是 Augment 自动注册工具的后端 API 服务。

✅ API 服务正常运行

📡 API 端点

POST /api/code - 获取邮箱验证码

请求示例：

curl -X POST "https://augment-register.vercel.app/api/code" \
  -H "Content-Type: application/json" \
  -d '{"email": "test@example.com"}'

响应示例：

{
  "success": true,
  "code": "123456",
  "message": "验证码获取成功"
}

🔧 使用说明

  1. 确保已配置环境变量 QQ_USER 和 QQ_PASS
  2. 使用油猴脚本调用此 API 获取验证码
  3. API 会自动连接 QQ 邮箱获取最新验证码
